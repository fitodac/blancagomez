== Description ==

Beautify your WordPress admin :) No settings just activate and enjoy a fresh WordPress admin experience!

== Installation ==

1. Upload the WpRedesigned plugin to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!, there are no settings for this.
